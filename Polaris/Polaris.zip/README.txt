Polaris Tutorial

Prerequisites:
- Fortnite 1.8
- DLL Injector
- Polaris DLL
- A Private Server

1 - Launch the Fortnite 1.8 client with the private server (not included, find yourself.)
2 - Go to the Battle Royale Lobby.
2 - Inject the Polaris DLL.
3 - Once it injects, press the PLAY button.
4 - Polaris will start loading, and once it's finished you will be placed on spawn island.

Congrats! You are now able to use Polaris.

-----

Extra

- Press the HOME button to display the Polaris GUI.
- Choose any skin you want in the lobby, and it will display in game
- Inject Console Unlocker for extra fun, but it is not required
